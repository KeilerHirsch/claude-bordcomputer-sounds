# Contributing to claude-bordcomputer-sounds

Thanks for your interest -- issues and pull requests are welcome.

## Ground rules
- Keep commits OPSEC-clean: your commit author email MUST be a GitHub noreply address
  (`<user>@users.noreply.github.com`). Set it locally: `git config user.email ...`.
- Match the existing style; keep each change focused.
- CI must pass before a PR can be merged.
